## 実行方法
Rコンソール上で、以下のように実行できる。

chapter02/09-add-CSS を実行する場合。
```R
> library(shiny)
> runGitHub("ShinyBook", "Np-Ur", subdir= "chapter02/09-add-CSS")
```

ただし、用いているライブラリをインストールしていない場合、エラーになるのでご注意ください。
